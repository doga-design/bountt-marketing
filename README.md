## What is Bountt?

Bountt is a group expense tracking app built for the group chat & high-trust friend groups. 
It's designed for friends, roommates, and anyone who's ever ended a night out with a 47-message thread trying to figure out who owes what.

The core philosophy: **log it, split it, forget it.** Bountt handles the math through smart-algorithm so you don't have to — and settles when it actually makes sense, not only when the math checks out.

---

## Core features

- **Frictionless cost logging** — two-tap entry with smart payer detection and group-aware splitting
- **Personal feed** — see your own expenses privately before sharing with the group
- **Draft flow** — log now, share when you're ready
- **Smart settlement & Tab-matching** — algorithm suggests settling when balances actually warrant it, not constantly
- **Deterministic avatars** — every member gets a consistent color and avatar across all groups
- **Group toggle** — switch between personal view and shared group feed instantly
- **No debt language** — the UI never uses words like "debt", "owe", or "balance due"
